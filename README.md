# ExchangeHub-Ecommerce-Web-Application-
ExchangeHub is a full-stack e-commerce platform that enables users to buy, sell, and exchange products through a secure, responsive interface. Optimized CRUD operations, role-based authentication, and dashboard performance to deliver an efficient user experience.
# ExchangeHub – OLX Clone

ExchangeHub is a PHP-based eCommerce marketplace inspired by OLX, where users can buy, sell, and exchange products. The platform provides a simple and user-friendly interface for listing items, browsing products, and managing user accounts.

## 🚀 Features

- User Registration & Login
- Secure Authentication using Sessions
- Add, Edit, and Delete Product Listings
- Image Upload for Products
- Browse Available Products
- Category-wise Product Management
- Admin Dashboard
- User Profile Management
- Responsive Design using Bootstrap
- MySQL Database Integration

## 🛠️ Tech Stack

- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: PHP
- Database: MySQL
- Server: XAMPP / WAMP

## ⚙️ Installation

1. Clone the repository.

```bash
git clone https://github.com/your-username/ExchangeHub.git
```

2. Move the project folder to the `htdocs` directory (XAMPP).

3. Import the MySQL database into phpMyAdmin.

4. Update database credentials in:

```
includes/connection.php
```

5. Start Apache and MySQL from XAMPP.

6. Open your browser and visit:

```
http://localhost/ExchangeHub
```
